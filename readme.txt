This CRM project is a Django-based web application designed for efficient client management. 
It incorporates user authentication, a responsive dashboard, and comprehensive functionalities 
for adding, viewing, updating, and deleting client information. Utilizing Bootstrap and custom 
styling ensures a visually appealing and consistent user interface. The system includes user-friendly 
forms for registration, login, and data manipulation, implementing Django forms and crispy forms for 
enhanced styling. Overall, the CRM website provides a seamless and visually attractive platform for 
users to manage their client data effectively.
